Firebase + validación

Este ejemplo utiliza Firebase como backend de persistencia de datos y sincroniza a los clientes en tiempo real (puedes probarlo abriendo varias pestañas en tu navegador). Además, realiza validación instantánea utilizando propiedades computadas y dispara transiciones CSS cuando se agregan/quitan elementos.
